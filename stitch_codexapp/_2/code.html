Can you write a python script to parse a CSV and calculate the average of the 'revenue' column?

Certainly. Here is a Python script using the built-in `csv` module to accomplish that.

python 
import csv

def calculate_average_revenue(file_path):
    total = 0
    count = 0
    with open(file_path, mode='r') as file:
        reader = csv.DictReader(file)
        for row in reader:
            if 'revenue' in row:
                total += float(row['revenue'])
                count += 1
    return total / count if count > 0 else 0
